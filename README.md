# 🌦️ Power BI Weather Forecast Dashboard

## 📌 Project Overview

This project showcases an **interactive Weather Forecast Dashboard** built using **Microsoft Power BI**, integrating real-time data from a weather API. The dashboard provides **current weather conditions, 7-day forecasts, and air quality insights** for multiple cities across India.

It is designed to deliver **data-driven weather insights** through a clean, modern, and highly interactive interface.

---

## 🚀 Key Features

### 🌍 Multi-City Analysis

* Supports multiple cities including:

  * Ajmer, Ujjain, Gurgaon, Bangalore, Mumbai, Amritsar
  * Puri, Tuljapur, Kolhapur, Indore, Noida, Lucknow, Pune
* Dynamic city selection using slicers
* Real-time updates based on selected location

---

### 🌡️ Current Weather Insights

* Displays:

  * Current temperature (°C)
  * Weather condition (Sunny, Rainy, etc.)
  * Last updated timestamp
* Clean card-based UI for quick readability

---

### 📅 7-Day Weather Forecast

* Line chart visualization of temperature trends
* Enables users to:

  * Identify temperature fluctuations
  * Compare weekly weather patterns

---

### 🌧️ Rain Probability Analysis

* Bar chart representing **daily chance of rain (%)**
* Helps in planning activities based on weather conditions

---

### 🌅 Sunrise & Sunset Tracking

* Displays:

  * Sunrise time
  * Sunset time
* Useful for travel and daily scheduling

---

### 🌬️ Atmospheric Conditions

Key environmental metrics presented using KPI cards:

* Humidity
* Wind Speed
* Visibility
* Pressure
* UV Index
* Precipitation

---

### 🌫️ Air Quality Monitoring (AQI)

* Gauge chart indicating AQI level (e.g., Moderate, Good, Poor)
* Detailed pollutant breakdown:

  * PM2.5
  * PM10
  * CO (Carbon Monoxide)
  * NO2 (Nitrogen Dioxide)
  * O3 (Ozone)
  * SO2 (Sulfur Dioxide)

---

## 🔗 Data Source

* Weather data is fetched using a REST API from:

  * `https://www.weatherapi.com/`
* Data includes:

  * Current weather
  * Forecast (7 days)
  * Air quality index

---

## ⚙️ Data Processing

### Power Query (ETL Process)

* Extracted JSON data from API
* Transformed nested structures into tabular format
* Cleaned and formatted columns for reporting
* Created relationships between datasets
